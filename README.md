# COS326_2048

A basic OCaml library project.

## Building

```bash
dune build
```

## Running

To run the library in a REPL:

```bash
dune utop
```

Then you can use the library:

```ocaml
open Cos326_2048;;
hello ();;
add 2 3;;
multiply 4 5;;
```

## Installing Dependencies

If you need to install dune:

```bash
opam install dune
```
